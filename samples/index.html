Map Projections

Most projections provided by D3 are created via d3.geo.projection and are configurable: you can rotate the globe, scale or transform the canvas, etc. These are my practice projections.

US States

A standard map of the United States using the Albers USA projection. Each state has been given a tooltip withe name of the state, as well as its abbreviation. More data can be added to the tooltip if desired.

US with Zoom

This map of the US is constructed in the same manner as the previous map, but adds a "click-to-zoom" feature. The race data is from the USA Today US Census API - 2010.

US Districts

A standard map of the United States using the Albers USA projection. Map data is combined with the 114th congress data to present each states congrssional districts. I will be combining this map with congressional representative data to make the map more interactive.

US Counties

A standard map of the United States using the Albers USA projection. Map data is combined with the county and state topojson data to present all US counties.

Individual States

Topojson provides state data for drawing each state. That data can be combined with county data to present all counties within a state. My plan is to add population, area, and other county-relevent data to the tooltips.

Unemployment Choropleth

A Cloropleth map is a map that uses differences in shading, coloring, or the placing of symbols within predefined areas to indicate the average values of a property or quantity in those areas.

This map shows (past) unemployment data.
